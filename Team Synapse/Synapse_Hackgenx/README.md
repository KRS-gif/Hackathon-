# 🚀 Project Overview

The **Smart Budget Allocation System** is an AI-driven decision-support platform designed to improve **transparency, efficiency, and accountability** in government budget distribution.

The system uses machine learning to predict optimal budget allocation for ministries and integrates **real-time dashboards, blockchain security, and mobile-based input systems**.

---

# 🎯 Problem Statement

Traditional budget allocation systems:
- Rely heavily on manual decision-making  
- Lack transparency and real-time monitoring  
- Do not use predictive intelligence  
- Are difficult to audit  

This project solves these issues using **AI + Blockchain + Real-Time Analytics**.

---

# ✨ Key Features

### 🤖 AI-Powered Budget Prediction
- Predicts optimal budget using historical and contextual data
- Uses AutoML (AutoGluon Tabular Model)
- Provides reasoning along with predictions

### 📊 Real-Time Dashboard
- Streamlit-based analytics dashboard
- Live visualization of inputs and predictions
- Real-time Firebase integration

### 📱 Mobile Application
- Users can submit ministry/project details through mobile
- Real-time prediction response
- Integrated with backend and blockchain

### 🔐 Blockchain Integration
- Final budget decisions are stored on blockchain
- Immutable and tamper-proof storage
- Ensures transparency and auditability

### ☁️ Firebase Integration
- Stores prediction results
- Stores stakeholder feedback
- Real-time updates across platforms

---


---

# 🔄 System Workflow

1. User enters ministry/project details via **mobile app or dashboard**
2. Flask backend sends data to the **ML prediction model**
3. Model generates:
   - Budget prediction
   - Reasoning
4. Results stored in **Firebase**
5. Dashboard displays **real-time analytics**
6. Final approved budget stored on **blockchain**
7. Stakeholders submit feedback (also stored in Firebase)
